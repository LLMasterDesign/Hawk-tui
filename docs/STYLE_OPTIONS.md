# Hawk-tui Motion + Style Options

## Boot / Loading styles
- `dot`
- `line`
- `miniDot`
- `jump`
- `pulse`
- `points`
- `globe`
- `moon`
- `hamburger`

Use with:
```bash
gum spin --spinner <style> --title "warming command library" -- sleep 1
```

## Suggested profile sets
1. `Operator`: `points` + orange + thick borders
2. `Calm`: `line` + muted cyan + minimal border
3. `Aggressive`: `pulse` + bright warnings + short refresh intervals
